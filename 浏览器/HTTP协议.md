# HTTP2 特点 (使用 TCP协议)

1. 使用二进制传输, 更高效, 更紧凑
2. 对报头压缩, 降低开销
3. 多路复用,一个网络连接实现并行请求
4. 服务器主动推送,减少请求延迟
5. 默认加密

# HTTP3 特点 (QUIC, 使用 UDP 协议)

1. 减少了握手的延迟
2. 多路复用,没有 TCP 的阻塞问题
3. 连接迁移, 当由 wifi 转移到 4G 时, 连接不会被断开
4. 没有队收阻塞问题
