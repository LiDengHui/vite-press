学习路径建议 ​

推荐按以下顺序阅读，由浅入深：

  1. 基础入门 → 掌握核心概念与基本使用
  2. 光照阴影 → 让场景有质感
  3. 物理引擎 → 给物体加入真实碰撞与重力
  4. 骨骼动画 → 角色与复杂动画
  5. 交互优化 → 性能调优与工程化
  6. 渲染管线 → 理解光照计算与追踪
  7. 后处理管线 → 帧缓冲、后处理效果、抗锯齿
  8. 高级渲染 → 体积渲染、阴影技术、WebGPU

快速开始 ​

如果你刚接触 Three.js，直接从这里开始：

bash
# 1. 创建 Vite 项目
npm create vite@latest my-3d-app -- --template vanilla

# 2. 安装 Three.js
cd my-3d-app
npm install three

# 3. 启动开发服务器
npm run dev
js
// main.js
import * as THREE from 'three';

const scene = new THREE.Scene();
const camera = new THREE.PerspectiveCamera(75, innerWidth / innerHeight, 0.1, 1000);
camera.position.z = 5;

const renderer = new THREE.WebGLRenderer({ antialias: true });
renderer.setSize(innerWidth, innerHeight);
document.body.appendChild(renderer.domElement);

const geometry = new THREE.BoxGeometry();
const material = new THREE.MeshStandardMaterial({ color: 0x00ff00 });
const cube = new THREE.Mesh(geometry, material);
scene.add(cube);

const light = new THREE.DirectionalLight(0xffffff, 1);
light.position.set(5, 5, 5);
scene.add(light);

function animate() {
    requestAnimationFrame(animate);
    cube.rotation.x += 0.01;
    cube.rotation.y += 0.01;
    renderer.render(scene, camera);
}
animate();

奇淫技巧：学习加速通道 ​

  • 快速试错: 每篇文章下的 ShaderToy 链接都可以点击直接运行，修改参数看效果，比阅读更直观
  • 交叉阅读: 难题去看"相关阅读"链接的文章，形成知识网络
  • 先做后学: 先跑通 Three.js 官方示例，再回来补充理论
  • 性能对比: 同一效果多换几个算法试（如 FXAA vs TAA），理解各有千秋
  • 代码画画: 写一个 shader，用变量控制第三个色彩通道，绘制调试图
