# npm 常用命令

| 命令               | 说明             |
| ------------------ | ---------------- |
| npm version patch  | 升级小版本       |
| npm version minor  | 升级中版本       |
| npm version major  | 升级大版本       |
| npm publish        | 版本包发布       |
| npm link           | 链接当前包到全局 |
| npm link <package> | 链接全局包到本地 |
