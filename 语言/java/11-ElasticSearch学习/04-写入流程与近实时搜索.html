еҶҷе…ҘжөҒзЁӢдёҺиҝ‘е®һж—¶жҗңзҙў вҖӢ

1. иҝҷжҳҜд»Җд№Ҳ вҖӢ

ES зҡ„еҶҷе…ҘдёҚжҳҜз«ӢеҲ»е°ұиғҪиў«жҗңзҙўеҲ°пјҢиҖҢжҳҜз»ҸиҝҮеҲ·ж–°е’Ңж®өеҗҲе№¶зӯүиҝҮзЁӢиҝӣе…ҘеҸҜжЈҖзҙўзҠ¶жҖҒгҖӮ
иҝҷе°ұжҳҜе®ғвҖңиҝ‘е®һж—¶жҗңзҙўвҖқзү№жҖ§зҡ„жқҘжәҗгҖӮ

2. дёәд»Җд№ҲйҮҚиҰҒ вҖӢ

еҰӮжһңдёҚдәҶи§ЈеҶҷе…Ҙй“ҫи·ҜпјҢе°ұеҫҲйҡҫи§ЈйҮҠдёәд»Җд№Ҳж•°жҚ®еҲҡеҶҷе…Ҙж—¶жҗңзҙўдёҚеҲ°пјҢжҲ–иҖ…дёәд»Җд№Ҳйў‘з№ҒеҲ·ж–°дјҡеҪұе“ҚжҖ§иғҪгҖӮ
иҝҷжҳҜдёҡеҠЎйў„жңҹе’ҢжҖ§иғҪжІ»зҗҶдёӯзҡ„й«ҳйў‘й—®йўҳгҖӮ

3. ж ёеҝғеҶ…е®№ вҖӢ

  • еҶҷе…Ҙ buffer
  • refresh
  • flush
  • segment
  • merge
  • иҝ‘е®һж—¶жҗңзҙўжңәеҲ¶

4. еӯҰд№ йҮҚзӮ№ вҖӢ

  • зҗҶи§ЈеҶҷе…Ҙе’ҢеҸҜжҗңзҙўдёҚжҳҜеҗҢдёҖж—¶еҲ»
  • зҗҶи§Ј refresh дјҡеҪұе“ҚеҶҷе…Ҙеҗһеҗҗ
  • зҗҶи§Ј segment еҗҲе№¶е’Ңиө„жәҗж¶ҲиҖ—зҡ„е…ізі»

5. еёёи§Ғй—®йўҳ вҖӢ

  • иҜҜд»ҘдёәеҶҷжҲҗеҠҹе°ұдёҖе®ҡиғҪз«ӢеҲ»жҗңеҲ°
  • йў‘з№Ғдё»еҠЁ refresh еҜјиҮҙжҖ§иғҪжҠ–еҠЁ
  • еҝҪз•ҘеҶҷе…Ҙе’ҢжҹҘиҜўзҡ„жқғиЎЎ

6. з»ғд№ е»әи®® вҖӢ

  • з”»дёҖеј  ES еҶҷе…ҘеҲ°еҸҜжҗңзҙўзҡ„жөҒзЁӢеӣҫ
  • и§ӮеҜҹ refresh й—ҙйҡ”еҸҳеҢ–еёҰжқҘзҡ„еҪұе“Қ
  • жҖ»з»“еҶҷе…Ҙе’ҢжҗңзҙўдёҖиҮҙжҖ§зҡ„дёҡеҠЎеӨ„зҗҶж–№ејҸ

7. иҮӘжөӢй—®йўҳ вҖӢ

  • дёәд»Җд№Ҳ ES жҳҜиҝ‘е®һж—¶иҖҢдёҚжҳҜејәе®һж—¶
  • refresh е’Ң flush еҲҶеҲ«еңЁеҒҡд»Җд№Ҳ
  • дёәд»Җд№Ҳйў‘з№Ғ refresh дјҡеҪұе“ҚжҖ§иғҪ
