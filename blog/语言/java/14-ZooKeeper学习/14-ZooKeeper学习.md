# ZooKeeper学习

ZooKeeper 是分布式系统里非常经典的协调组件。  
学习 ZooKeeper 的重点，不只是会连上去读写节点，更重要的是理解数据模型、Watcher、会话机制、一致性协议和协调场景。

## 1. 学习定位

这一组内容用于补充 Java 分布式系统中的协调能力认知。  
目标是让你理解 ZooKeeper 在注册发现、配置管理、分布式锁和选主中的价值与边界。

## 2. 学习重点

- 理解 znode 模型和节点类型
- 理解会话和 Watcher 的行为特点
- 理解 ZooKeeper 的一致性基础
- 理解选主和协调场景的实现思路
- 理解运维问题为什么会直接影响上层服务

## 3. 建议顺序
